Set_printoptions#

Numpy outputs can be configured through set_printoptions function. Here is described with examples some options.

For more information, see the corresponding page in the official documentation.

linewidth#

How many characters will be displayed on each line, the default is 75. [ and \n are also counted. That‚Äôs why you always get 3 characters less from the array. The result seems to be approximate, because the specified linewidth may not be a multiple of the number of matrix symbols. Moreover, strings are different - minuses, and the value of the precision argument will affect the length of strings.

Here are a few examples of printing arrays using different values for the linewidth parameter. They show how arrays look in the output and then calculate the number of symbols in string representations.

linewidth = 40

np.set_printoptions(

    # be able to see the whole array
)
print(arr)

suppress#

Determines whether the exponential form of number recording will be used. In the case of True, numbers will always be printed in the usual format, abbreviated according to the value of the precision argument. In the case of False, if the array contains a number less than 0.0001 or the ratio of the maximum and modulus of the minimum element of the array is more than 1000, the whole array will be printed in exponential form. The default is False.

One small number#

Here we will consider an example where just one small number (<0.0001) will make numpy use the scientific power of the number.

unavailible_number = np.array([0.000099])
availible_number = np.array([0.0001])

np.set_printoptions(
    suppress = True,
    linewidth = 20, # —ç—Ç–æ –¥–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã —á–∏—Å–ª–∞
    # –ø–æ–º–µ—â–∞–ª–∏—Å—å –≤ –≤—ã–≤–æ–¥
    precision = 8 # —ç—Ç–æ —á—Ç–æ–±—ã —á–∏—Å–ª–∞, –∫–æ—Ç–æ—Ä—ã–µ —è —É–∫–∞–∑–∞–ª
    # –Ω–µ –æ–∫—Ä—É–≥–ª–∏–ª–∏—Å—å –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏
)

print("=================suppress = True================")
print("availible number", availible_number)
print("unavailible number", unavailible_number, end = "\n\n\n")

np.set_printoptions(suppress = False)
